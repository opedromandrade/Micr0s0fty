# 🛠️ My Window$ Post Install Script Collection

## 🎉 What is this?  
A mischievously **privacy‑first** collection of PowerShell scripts that **tame Winget**, **shush AI assistants**, and make Windows act like a secret‑agent‑level fortress. Think of it as giving your PC a *“Do Not Disturb”* badge—only cooler.  

## 📦 What’s Inside?  
| 📂 Folder | 🛠️ Purpose | 🤖 What It Does |
|----------|------------|----------------|
| `winget‑helpers/` | Streamline package installs | Bulk‑install, auto‑approve, and clean up leftovers so you never have to type `y` again. |
| `ai‑silencer/` | Reduce AI “listening” | Disables Cortana, disables Windows Copilot telemetry, and removes preview AI bundles. |
| `privacy‑armor/` | Harden Windows privacy | Tweaks telemetry, disables unnecessary background services, and configures firewall rules for a ninja‑level low‑profile. |
| `docs/` | Your cheat‑sheet | Markdown guides, quick‑start tutorials, and a FAQ that answers “Why am I still being watched?”  

## 🚀 Getting Started (in 3 Easy Steps)  
1. **Clone the repo** – `git clone https://github.com/your‑name/winget‑privacy.git`  
2. **Open PowerShell as Admin** – because we *need* those elevated privileges.  
3. **Run the master script** – `.\run‑all.ps1` and watch the magic happen. 🎩  

> **Pro tip:** Run it on a fresh Windows install for maximum effect.
